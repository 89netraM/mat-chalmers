mat-chalmers
============

Lunch menus on Campus Johanneberg

## Hacking

- `git clone ...`
- `cd ...`
- copy the stack file for your preffered GHC version
  (`stack-7.10.3.yaml` or `stack-7.8.4.yaml`) to `stack.yaml`.
- `stack build`
